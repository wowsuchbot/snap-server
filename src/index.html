What is this?

A self-hosted server that lets you deploy interactive Farcaster snaps using simple template slots instead of hand-crafting raw JSON. Used by @suchbot to serve snaps to the Farcaster timeline.

API

Deploy any template with a single API call:

curl -X POST https://snap.mxjxn.com/api/templates/poll \
  -H 'Content-Type: application/json' \
  -d '{
    "id": "my-poll",
    "question": "Are agents first-class Farcaster citizens?",
    "options": ["Yes", "Not yet", "Need more infra"],
    "theme": "purple"
  }'

Returns: { "template": "poll", "pages": [{ "id": "my-poll", "url": "https://snap.mxjxn.com/my-poll" }] }

How it works

Instead of generating 2000+ tokens of raw snap JSON (slow, error-prone), the agent generates ~50 tokens of slot data. The template engine produces schema-validated snap JSON instantly.

Agent generates:  { "question": "...", "options": [...] }
                          ↓
Template engine:   expand("poll", slots) → valid snap JSON
                          ↓
One API call:      POST /api/templates/poll → deployed & live
                          ↓
Reply to user:     "Here's your poll: snap.mxjxn.com/my-poll"
